In general, you can describe base substitutions by using
[variation](/ddbj/features-e.html#variation) feature with 
 /[replace](/ddbj/qualifiers-e.html#replace) and
 /[note](/ddbj/qualifiers-e.html#note) qualifiers.    

In case of using DDBJ Nucleotide Sequence Submission System, select 'other' for template.    
About format of feature annotation, see 
[F01) polymorphism and variation](/ddbj/example-e.html#F01) at 
[Description Examples of Sequence Data](/ddbj/example-e.html).
