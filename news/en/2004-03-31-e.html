  • New option of search and analysis service
The search and analysis services by E-mail, a new option "email_request_id" which can request Request ID notifying mail, will start from May 10, 2004.
     Services: FASTA, BLAST, SSEARCH, S&W SEARCH, ClustalW
     Example (request): email_request_id 1
When the request of the Request ID notifying mail is specified by this option, request ID is sent to the user by e-mail. The user can confirm the Internet congestion at Traffic information page on Search and Analysis using this request ID. When not specified, the mail is not sent. For ClustalW, Request ID had been notified to all users so far.But, hereafter, please note that you need to specify "email_request_id" option like as another homology services.
