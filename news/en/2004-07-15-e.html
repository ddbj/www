  • Inconsistency of some entries of DDBJ Release 58
Inconsistency was found in some entries of the latest DDBJ Release 58 (released on June 30, 2004).
Nature of inconsistency:
Description of "BASE COUNT" line was not included in some data.
Measures implemented:
Defected data were corrected, and released again on July 9.
List of corrected files:
We DDBJ regret our mistake.
[Note on "BASE COUNT" line]
  Although GenBank had dropped the BASE COUNT line from their flat file format from GenBank Release 138 (Oct. 2003), we DDBJ still keep the BASE COUNT line in DDBJ flat file format, because we believe that this information is useful, such as computing the GC contents.
