  • ARSA での DDBJ，DAD 以外の DB 検索の一時休止　
ARSA は DDBJ が WWW ならびに Web API で提供している高速なキーワード検索サービスです。検索対象データベースのうち，DDBJ，DAD 以外の DB 検索に不具合が発生いたしました。詳細は以下の通りです。
  • 停止期間： 2009年11月3日（火）11時30分より11月4日（水）13時30分まで
  • 停止サービス： ARSA の DDBJ，DAD 以外（全18DB）検索の一時休止
  • 備考： 上記停止期間において，ARSA で DDBJ，DAD の検索は可能な状況でした。
  • 対応状況：
      ・11月4日13:30現在，10月25日時点のデータを検索が可能な状況に復旧しております。
      ・最新のデータに復旧後，再度アナウンスさせて頂きます。

ユーザの皆様にはご迷惑をお掛けいたしました事を深くお詫び申し上げます。
復旧作業は終了しました。（11月05日 13時30分）
