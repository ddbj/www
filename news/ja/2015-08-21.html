韓国特許庁(Korean Intellectual Property Office: KIPO)に特許出願された塩基配列データおよびアミノ酸配列データが公開されました。
今回の登録データより、RNAデータが公開されます。またCOMMENTブロックでは、新規にCC行、FH行、FT行の追加が行われています。

1) アクセッション番号
塩基配列データ(77,591件)： DI409117-DI486707　　内訳 (DNAデータ: 47,242件, RNAデータ: 30,349件)
アミノ酸配列データ(46,456件)：DI746460-DI792915

公開データは、getentryでのアクセッション番号検索、ARSAでのキーワード検索が可能です。



2) COMMENTブロックの説明
COMMENTブロックのTY行の後に、CC行、FH行、FT行が追加されました。
各行のヘッダーは、下記の内容が記載されています。

------------------------------------------------------------------------------------------      Header       Description contents------------------------------------------------------------------------------------------COMMENT  　 KN    　　Sequence ID            AN        Application number            AD        Application date            PN        Publication number            PD        Publication date            AT        Application title            AI        Inventors (bar-separated)            AA        Applicants (bar-separated)            PR        Priority number=Priority date (bar-separated)            OS        Organism name            TY        Sequence type (DNA, RNA, PRT)            CC        Comment            FH        Fixed character string of "Key 　Location/Qualifiers"            FT        Feature/qualifier information------------------------------------------------------------------------------------------</code>


(新規追加行の説明)
CC行: 配列内容や出願人のコメントが記載されます。データによっては、省略される場合があります。
FH行: “Key 　Location/Qualifiers” の固定文字列が、全件に記載されます。
FT行: Feature/Qualifier情報が記載されます。データによっては、省略される場合があります。
