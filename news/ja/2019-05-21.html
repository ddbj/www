  • GEA が Scientific Data の Recommended Data Repositories に収載されました

生命情報・DDBJ センターが運営している遺伝子発現やエピジェネティクス等の機能ゲノミクスデータのための公的データベースである Genomic Expression Archive (GEA) が Scientific Data の Recommended Data Repositories に収載されました。 GEA はマイクロアレイとシークエンスによるデータをそれぞれ MIAME と MINSEQE ガイドラインに従い MAGE-TAB 形式で受け付けています。 GEA は Experiment に対して E-GEAD-n、アレイデザインに対して A-GEAD-n 形式のアクセッション番号を発行しています。

GEA のメタデータは EBI Omics DI でインデックスされる予定です。
