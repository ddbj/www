<html><a href="http://arsa.ddbj.nig.ac.jp/top-j.html">ARSA</a> は DDBJ が WWW ならびに Web API で提供している高速なキーワード検索サービスです。検索対象データベースのうち，DDBJ，DAD 以外の DB 検索に不具合が発生いたしました。詳細は以下の通りです。

<ul>
    <li><b>停止期間：</b> 2009年11月3日（火）11時30分より11月4日（水）13時30分まで</li>
    <li><b>停止サービス：</b> ARSA の DDBJ，DAD 以外（全18DB）検索の一時休止</li>
    <li><b>備考：</b> 上記停止期間において，ARSA で DDBJ，DAD の検索は可能な状況でした。</li>
    <li><b>対応状況：</b><br>  <b>・</b>11月4日13:30現在，10月25日時点のデータを検索が可能な状況に復旧しております。<br>  <b>・</b>最新のデータに復旧後，再度アナウンスさせて頂きます。</li>
</ul>

<p>ユーザの皆様にはご迷惑をお掛けいたしました事を深くお詫び申し上げます。<br>
    <font color="#ff0000">復旧作業は終了しました。（11月05日 13時30分）</font>
</p>
