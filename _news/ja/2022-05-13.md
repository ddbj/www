2022年6月9日 本不具合は解消されました。    

NCBI の locus tag prefix 予約 api に不具合が生じており、DDBJ センターの BioProject 登録システムで locus tag prefix を予約しようとすると submit が失敗する状況になっております。   
locus tag prefix が必要な方は BioProject では記入欄を空欄にして予約せず、BioSample の locus_tag_prefix 属性に希望するタグを記入して登録してください。  
ご迷惑をおかけいたしますが、何卒宜しくお願い致します。
