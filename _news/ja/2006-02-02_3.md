<html>1月31日から2月2日にかけて，農業生物資源研究所から登録されたイネ（<i>Oryza sativa</i> (japonica cultivar-group)）の EST データ78万エントリが公開されました。<br>アクセション番号は以下の通りです。

<ul>
    <li>CI000001-CI778739 </li>
</ul>

<p>これらは1月31日-2月2日分の新着データとして DDBJ より公開されており， <a href="https://ddbj.nig.ac.jp/public/ddbj_database/mass/rice_EST">FTP サイト</a> から取得することができます。<br>- <a href="https://ddbj.nig.ac.jp/public/ddbj_database/mass/rice_EST/rice_EST1.seq.gz">rice_EST1.seq.gz</a>（CI000001-CI250000：圧縮サイズ83MB）<br>- <a href="https://ddbj.nig.ac.jp/public/ddbj_database/mass/rice_EST/rice_EST2.seq.gz">rice_EST2.seq.gz</a>（CI250001-CI500000：圧縮サイズ76MB）<br>- <a href="https://ddbj.nig.ac.jp/public/ddbj_database/mass/rice_EST/rice_EST3.seq.gz">rice_EST3.seq.gz</a>（CI500001-CI778739：圧縮サイズ79MB）</p>
