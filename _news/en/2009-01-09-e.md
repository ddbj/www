<a href="http://arsa.ddbj.nig.ac.jp/top-e.html">ARSA</a> is a high-speed data retrieval system provided by DDBJ via WWW and Web API. DDBJ and DAD database search in ARSA is unavailable in the following schedule for update each of released data (DDBJ Release 76.0, DAD Release). Details are as follows:

<ul>
    <li>Date &amp; Time. Jan. 16, 2009 (Fri) 9:00 - 18:00 (We will announce on this page when it resumes.):</li>
    <li>Unavailable services:<br>  DDBJ and DAD search in ARSA (another 18DBs are searchable. Search results do not contain DDBJ and DAD entries)<br>  ARSA reference function in TX Search<br>  DNA/Protein/AllDBs search at the search box of the upper part of DDBJ HP.(another DB search are available)</li>
</ul>
