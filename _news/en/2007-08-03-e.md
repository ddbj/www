<dd>DDBJ completed the trial service of <a href="http://arsa.ddbj.nig.ac.jp/top-e.html" target="_blank">ARSA</a> and released it officially.
<dd>The features of ARSA are as follows:<br>

    <ul><br>
        <li><br>ARSA provides one-stop query to 23 life science databases including DDBJ International Nucleotide Sequence Database.

            <p></p>
        </li>
        <li>ARSA will return a quick response to any query regardless of the complexity of the query, the number of hits to the query and the target database(s)

            <p></p>
        </li>
        <li>In the case of DDBJ, you can define search conditions in details by use of any combination of <a href="/ddbj/flat-file-e.html">Features/Qualifiers</a> of the DDBJ flat file.

            <p></p>
        </li>
        <li><a href="http://www.xml.nig.ac.jp" target="_blank">Application Programming Interface (API)</a> is prepared so that you can call some functions of ARSA from your Java or Perl programs in your computer. <br></li>
    </ul>
<dd>DDBJ added the following new functions based on the users' comments.

    <table border="0">
        <tr>
            <td valign="top">

                <ul>
                    <li>The search results can be downloaded across databases (1)</li>
                    <li>The database can be selected in the reslut pages (3)</li>
                    <li>The result items shown in the result pages can be<br> specified in the all databases (2, 4)</li>
                    <li>The number of the records per page can be changed (5)</li>
                    <li>The links of page jump are added in the result pages.<br></li>
                </ul>
            </td>
            <td valign="top"><a href="{{ site.baseurl }}/assets/images/news/ARSA07080302.gif"><img src="{{ site.baseurl }}/assets/images/news/ARSA07080303.gif" width="194" height="204" align="right" alt="ARSA"></a></td>
        </tr>
    </table>

    <p>Another Changes<br></p>

    <ul>
        <li>The extension of file which can be downloaded with TSV format was changed from ".csv" to ".txt".</li>
        <li>"Narrow Search " changed its name to "Refine Search".</li>
        <li>SimpleQuery was removed.</li>
        <li>HTTP download was abolished. You can download the results from anonymous FTP.<br></li>
    </ul>
<dd><a href="http://arsa.ddbj.nig.ac.jp/top-e.html" target="_blank">ARSA</a> will continue to be enhanced, if you make comments. So, please visit <a href="http://arsa.ddbj.nig.ac.jp/top-e.html" target="_blank">ARSA</a> and click "Your Comment" to help us to improve ARSA to help you.
<dd>(*) ARSA means "Here you go" in Japanese.</dd>
