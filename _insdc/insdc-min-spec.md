INSDCは、加盟するすべてのデータベース間で共有される塩基配列データの一貫性、相互運用性、および信頼性を確保するため、一連の Minimal specifications を策定しました。

2025年に INSDC Implementation Committee によって正式に定められたこれらの仕様は、既存の標準を現代のデータワークフローに適合するよう再構成したものです。統一されたデータモデルと明確な受理基準を定義することにより、データの検証および交換の方法を整理・統合しました。これらの文書は「living documents（継続的に更新される文書）」であり、技術進歩や、拡大し続ける世界の科学コミュニティの多様性に合わせて、柔軟に進化させていくことを目的としています。

## Minimal specifications {#spec}

* [Analysis](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Analysis/Current%20version/Analysis%20Minimum%20Specification%20v1.0.docx.pdf)
    Secondary data files that are derived from computational workflows or provide supplemental biological context to INSDC nucleotide data.
* [Annotation](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Annotations/Current%20version/Annotation%20Minimum%20Specification%20v1.0.docx.pdf)
    Description and location of the biological features in the cited sequence.
* [Assembly](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Assembly/Current%20version/Assembly%20Minimum%20Specification%20v1.0.docx.pdf)
    Description of the set of assembled sequences and related metadata that identifies the chromosomes, unlocalized sequences, and unplaced sequences that represent a genome.
* [Compressed reads](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Compressed%20Reads/Current%20version/Compressed%20Reads%20Minimum%20Specification%20v1.0.docx.pdf)
    Sequence read files without base level qualities provided in INSDC specified formats.
* [Experiment](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Experiment/Current%20version/Experiment%20Minimum%20Specification%20v1.0.docx.pdf)
    Collected sequencing experimental information (methods, platform, library, and machine configuration).
* [Package-checklist](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Package-Checklist/Current%20version/Package-Checklist%20Minimum%20Specification%20v1.0.docx.pdf)
    A predetermined set of metadata requirements for sample registration.
* [Project](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Project/Current%20version/Project%20Minimum%20Specification%20v1.0.docx.pdf)
    A collection of biological data and description of research effort related to a single initiative.
* [Raw reads](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Raw%20Reads/Current%20version/Raw%20Reads%20Minimum%20Specification%20v1.0.docx.pdf)
    Sequence read files containing base level information provided in INSDC specified formats.
* [Sample](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Sample/Current%20version/Sample%20-%20Minimum%20Specification%20v1.0.docx.pdf)
    The description of the biological source materials used in experimental assays.
* [Sequence](https://github.com/INSDC-Repo/INSDC/blob/main/INSDC%20Minimal%20Specifications/Sequence/Current%20version/Sequence%20minimum%20standards%20v1.0.docx.pdf)
    Assembled nucleotide sequences.
