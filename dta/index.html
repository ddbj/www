DDBJ Trace Archive

DDBJ Trace Archive (DTA) は，配列決定プロジェクトからのシングルパスリードの DNA sequence chromatograms (traces), base calls, quality estimates のためのデータベースです。 DTA は International Nucleotide Sequence Database Collaboration (INSDC) のメンバーであり，NCBI Trace Archive と EBI Trace Archive との国際協力の下，データベースを運営しています。 DTA から公開されたデータは NCBI Trace Archive で検索，取得することができます。
