Definition of Terms

Web submission: Data submission to DDBJ via Nucleotide Sequence Submission System
MSS: Data submission to DDBJ via Mass Submission System
Submitter: Number of groups of submitters
Session: Number of times to load submitted data into DDBJ
Japan: Submitted from Japanese groups
other: Submitted from groups in countries other than Japan
